# Device Readiness Checklist

Before installation begins, verify that schools meet the minimum device and connectivity conditions required for consistent Giga Meter data collection.

> [!IMPORTANT]
> Giga Meter currently runs on Windows only. Tablets and Android devices cannot run the app.

---

## Per-school checklist

For each school targeted for installation, confirm the following:

### Device requirements

- [ ] **Windows OS** — Windows 7 or higher (desktop or laptop)
- [ ] **Free disk space** — at least 500 MB available
- [ ] **Regular use** — device is used daily or near-daily (a device that is rarely powered on will collect very little data)
- [ ] **Dedicated school connection** — device connects exclusively to the school's internet, not a mobile hotspot or personal router
- [ ] **Admin rights available** — the person installing Giga Meter can approve the installation (admin password or IT approval in hand)

### Connectivity requirements

- [ ] School internet connection is functional and stable enough to complete an installation
- [ ] Device can reach the internet at the time of installation

---

## Number of devices per school

Install Giga Meter on **at least two computers per school**. More devices mean:
- More frequent measurements
- Redundancy if one device is switched off, repaired, or reformatted
- A more reliable picture of the school's connectivity over time

> [!TIP]
> If a computer is replaced or reformatted, Giga Meter must be reinstalled. Build this into your maintenance plan by assigning a school-level focal point responsible for reinstallation.

---

## Common blockers

| Blocker | What to do |
|---|---|
| School only has tablets or Chromebooks | Flag to Giga — Android support is in development. Consider manual data collection in the interim. |
| No Windows device available | Identify if the school has any Windows computer (even in an office) that stays powered on during school hours |
| Computers not regularly powered on | Discuss with school principal — measurements only run when the device is on and connected |
| No admin rights | Coordinate with MoE IT department to grant temporary admin access for installation |
| Internet connection down at time of visit | Reschedule or use a different device; note in the rollout tracker |

---

## Recording device status

Before the rollout, collect device status at the school level. A simple spreadsheet or form covering:

- School name and ID
- Number of Windows devices available
- Average hours powered on per day
- Confirmed admin rights: yes / no
- Notes (e.g., device is shared across classrooms, only available after 15:00)

Share this with the Giga team before the rollout begins. It informs deployment monitoring and helps set realistic expectations on how quickly schools will start reporting.
