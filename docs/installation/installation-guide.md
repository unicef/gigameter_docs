# Installation Guide

Before you start, confirm your device meets the [System Requirements](system-requirements.md).

***

### Step-by-step installation

#### Step 1 - Download Giga Meter

{% columns %}
{% column %}
Visit [meter.giga.global](https://meter.giga.global/) and click **Download** in the upper right corner.

If you have visited the page before, hard refresh first so you get the latest version:

{% endcolumn %}

{% column %}
<figure><img src="https://448102781-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FnO4TCnw5Pvr77OIRKx3h%2Fuploads%2Fgit-blob-2368e25df75371596e81b244374d293a095e4b4a%2Fapp-school-id-entry.png?alt=media" alt="Giga Meter download page with download button highlighted"><figcaption></figcaption></figure>
{% endcolumn %}
{% endcolumns %}

#### Step 2 - Open the downloaded file

{% columns %}
{% column %}
Click the file to open it. It may appear in your browser's bottom bar, the downloads icon, or your **Downloads** folder.

If Windows shows a **"Windows protected your PC"** SmartScreen warning, click **More info** then **Run anyway**. The app is published by UNICEF and safe to install.
{% endcolumn %}

{% column %}
<figure><img src="https://448102781-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FnO4TCnw5Pvr77OIRKx3h%2Fuploads%2Fgit-blob-17b300139c0cd35ff32201e35f920cb1dfd5faae%2Fsetup-smartscreen.png?alt=media" alt="Windows SmartScreen warning: click More info then Run anyway"><figcaption></figcaption></figure>
{% endcolumn %}
{% endcolumns %}

#### Step 3 - Check the version

If a warning about an older version appears, go back to step 1 and reinstall the latest version.

#### Step 4 - Allow installation

{% columns %}
{% column %}
If prompted by a **User Account Control** dialog, click **Yes**. The publisher is UNICEF. This opens the Setup Wizard.
{% endcolumn %}

{% column %}
<figure><img src="https://448102781-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FnO4TCnw5Pvr77OIRKx3h%2Fuploads%2Fgit-blob-cb8ac7281c5fbe40719ecd19a7aa7d5e675e7147%2Fsetup-uac-prompt.png?alt=media" alt="Windows UAC dialog asking to allow Giga Meter to make changes - click Yes"><figcaption></figcaption></figure>
{% endcolumn %}
{% endcolumns %}

#### Step 5 - Choose installation scope

{% columns %}
{% column %}
Select **"Anyone who uses this computer (all users)"** and click **Next**. This makes the app available to all user accounts on the device.
{% endcolumn %}

{% column %}
<figure><img src="https://448102781-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FnO4TCnw5Pvr77OIRKx3h%2Fuploads%2Fgit-blob-a8157ead9c975b7d9774c86e64e310db2ae50e7e%2Fsetup-installation-scope.jpg?alt=media" alt="Setup wizard: Choose Installation Options - select Anyone who uses this computer"><figcaption></figcaption></figure>
{% endcolumn %}
{% endcolumns %}

#### Step 6 - Choose destination folder

{% columns %}
{% column %}
The default folder (`C:\Program Files\unicef-pdca\Giga Meter`) is recommended. Click **Install**.
{% endcolumn %}

{% column %}
<figure><img src="https://448102781-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FnO4TCnw5Pvr77OIRKx3h%2Fuploads%2Fgit-blob-5dc9afbcfeab48ba478562cd8e9e4b5f3046e863%2Fsetup-install-location.jpg?alt=media" alt="Setup wizard: Choose Install Location with default destination folder"><figcaption></figcaption></figure>
{% endcolumn %}
{% endcolumns %}

#### Step 7 - Finish the setup

{% columns %}
{% column %}
Click **Finish** on the Setup Wizard. Leave **"Run Giga Meter"** checked to open the app immediately.
{% endcolumn %}

{% column %}
<figure><img src="https://448102781-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FnO4TCnw5Pvr77OIRKx3h%2Fuploads%2Fgit-blob-1e4aae0986e6dd636a73b5f5bbca3bd6475e0fa1%2Fsetup-finish.png?alt=media" alt="Setup wizard: Completing Giga Meter Setup - click Finish"><figcaption></figcaption></figure>
{% endcolumn %}
{% endcolumns %}

#### Step 8 - Open Giga Meter

{% columns %}
{% column %}
If the app did not open automatically, find the **Giga Meter** shortcut on your Desktop and double-click it.
{% endcolumn %}

{% column %}
<figure><img src="https://448102781-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FnO4TCnw5Pvr77OIRKx3h%2Fuploads%2Fgit-blob-80379d507c508549872be53b4f2d5e2a36173cd7%2Fsetup-desktop-shortcut.png?alt=media" alt="Windows desktop showing the Giga Meter shortcut icon"><figcaption></figcaption></figure>
{% endcolumn %}
{% endcolumns %}

***

### Registration

#### Step 9 - Start registration

{% columns %}
{% column %}
Follow the registration steps shown in the app. Click **Next** to move through the onboarding screens.

The app will remind you to:

* Install only on computers connected exclusively to the school's internet
* Register up to two devices per school
{% endcolumn %}

{% column %}
<figure><img src="https://448102781-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FnO4TCnw5Pvr77OIRKx3h%2Fuploads%2Fgit-blob-f0b15a149b0d7568a75c7a5ef5412a403b19c022%2Fapp-welcome.png?alt=media" alt="Welcome to Giga Meter screen"><figcaption></figcaption></figure>
{% endcolumn %}
{% endcolumns %}

#### Step 10 - Select your language

{% columns %}
{% column %}
To change the language, click **English** in the top right corner and select from the list.
{% endcolumn %}

{% column %}
<figure><img src="https://448102781-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FnO4TCnw5Pvr77OIRKx3h%2Fuploads%2Fgit-blob-73c5add60705194376e4cb59633da74f0036ef84%2Fapp-language-select.png?alt=media" alt="Language selection screen"><figcaption></figcaption></figure>
{% endcolumn %}
{% endcolumns %}

#### Step 11 - Accept the policies

{% columns %}
{% column %}
Read the policies before accepting. Accept the **Data Privacy Policy** and **Data Sharing License** by checking the box.

Click **Start Registration** to continue.
{% endcolumn %}

{% column %}
<figure><img src="https://448102781-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FnO4TCnw5Pvr77OIRKx3h%2Fuploads%2Fgit-blob-b25fc96ba95cc4e03cb3cf9996871d6cf4cce662%2Fapp-accept-policies.png?alt=media" alt="Accept policies screen with Start Registration button"><figcaption></figcaption></figure>
{% endcolumn %}
{% endcolumns %}

#### Step 12 - Confirm your country

{% columns %}
{% column %}
Your country should be detected automatically. Click **Confirm** to continue.

{% hint style="warning" %}
If the detected country is incorrect, select the correct one from the dropdown and click **OK**. Then click **Confirm** to continue.
{% endhint %}
{% endcolumn %}

{% column %}
<figure><img src="https://448102781-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FnO4TCnw5Pvr77OIRKx3h%2Fuploads%2Fgit-blob-aa70317fda4ccd6fa0b6e83799e331c31e9bdcc5%2Fapp-country-detected.png?alt=media" alt="Country automatically detected screen"><figcaption></figcaption></figure>
{% endcolumn %}
{% endcolumns %}

#### Step 13 - Country verification errors

{% columns %}
{% column %}
**"It looks like the country you selected may not be accurate"**\
Double-check your country selection. If it is correct and the message still appears, this may be caused by a VPN or an IP address signalling a different country. You can continue by clicking **Confirm**.

**"Sorry, Giga Meter is not available in \[...]"**\
Verify your selected country is correct. If the issue persists, contact your designated Giga Meter focal point at UNICEF or the school administrator who guided the installation.
{% endcolumn %}

{% column %}
<figure><img src="https://448102781-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FnO4TCnw5Pvr77OIRKx3h%2Fuploads%2Fgit-blob-9c624b411df131d3e565d871485595a61d2639c5%2Fapp-country-warning.png?alt=media" alt="Country verification warning screen"><figcaption></figcaption></figure>
{% endcolumn %}
{% endcolumns %}

#### Step 14 - Enter your school ID

{% columns %}
{% column %}
Enter the school ID, a unique identifier provided by the government, and click **Search ID**. Select your school from the results by clicking **Select**.

{% hint style="success" %}
If you are unsure of your school ID, check with your school administrator or IT department.
{% endhint %}
{% endcolumn %}

{% column %}
<figure><img src="https://448102781-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FnO4TCnw5Pvr77OIRKx3h%2Fuploads%2Fgit-blob-2368e25df75371596e81b244374d293a095e4b4a%2Fapp-school-id-entry.png?alt=media" alt="Enter school ID screen"><figcaption></figcaption></figure>

<figure><img src="https://448102781-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FnO4TCnw5Pvr77OIRKx3h%2Fuploads%2Fgit-blob-2d2f2654c9ae10bb4225961e509606d7c0d6489f%2Fapp-school-id-results.png?alt=media" alt="School search results with Select button"><figcaption></figcaption></figure>
{% endcolumn %}
{% endcolumns %}

#### Step 15 - Confirm the school name

{% columns %}
{% column %}
Double-check the information shown. If the school name is correct, click **YES** to confirm.

{% hint style="warning" %}
Once you click **YES**, changing your school ID will require uninstalling and reinstalling the app.
{% endhint %}
{% endcolumn %}

{% column %}
<figure><img src="https://448102781-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FnO4TCnw5Pvr77OIRKx3h%2Fuploads%2Fgit-blob-0822d3cc7add4824dea6b646d74c3ea30e9340a7%2Fapp-school-confirm.png?alt=media" alt="Confirm school information screen"><figcaption></figcaption></figure>
{% endcolumn %}
{% endcolumns %}

#### Step 16 - First connectivity test

{% columns %}
{% column %}
Your school is now registered. The app automatically runs the first connectivity test to confirm registration was completed successfully. This may take a moment.
{% endcolumn %}

{% column %}
<figure><img src="https://448102781-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FnO4TCnw5Pvr77OIRKx3h%2Fuploads%2FUx2ul3OD4Zhr2Okr6G8f%2FGroup%208809.png?alt=media&#x26;token=3cb5f785-ff92-4cf0-8b8c-1a1338c109c5" alt="Registration complete, running first connectivity test"><figcaption></figcaption></figure>
{% endcolumn %}
{% endcolumns %}

#### Step 17 - Installation complete

{% columns %}
{% column %}
Your installation is complete.

If the test is unsuccessful, try again later when the internet connection improves. If tests fail repeatedly, take a screenshot and share it with the administrator who guided you through the installation.

{% hint style="info" %}
Your school connectivity will be measured daily and data will be sent to Giga automatically. For accurate data, keep the computer turned on and connected to the school's internet for as many hours as possible each day.
{% endhint %}
{% endcolumn %}

{% endcolumn %}
{% endcolumns %}
